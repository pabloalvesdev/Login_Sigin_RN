# Como executar o projeto

## Front end
Pré-requisitos:
- npm ou yarn
- React Navigation V4 (Rotas)
- React Native

```bash
# clonar repositório
git clone https://github.com/pabloalvesdev/Login_Sigin_RN.git

# entrar na pasta do projeto
cd Login_Sigin_RN

# instalar dependências
npm install

# executar o projeto
npm start
```
